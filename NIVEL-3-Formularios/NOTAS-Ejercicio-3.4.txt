1️⃣ Objetivo del ejercicio

Crear un formulario de inicio de sesión básico que permita al usuario ingresar usuario y contraseña, valide los campos obligatorios y muestre un mensaje de bienvenida o error sin recargar la página.

2️⃣ Semántica

<form> → Contenedor del formulario.

<label> → Etiquetas de los campos.

<input> → Campos de usuario y contraseña.

<button> → Botón para enviar.

<div id="mensaje"> → Contenedor para mostrar mensajes dinámicos (éxito o error).

3️⃣ Pseudocódigo
Inicio
Mostrar formulario:
    - Input Usuario
    - Input Contraseña
    - Botón Ingresar

Al hacer clic en "Ingresar":
    Capturar valores de Usuario y Contraseña
    Si Usuario o Contraseña está vacío:
        Mostrar mensaje: "Debe completar todos los campos"
    Sino
        Si Usuario == "admin" y Contraseña == "1234":
            Mostrar mensaje: "Bienvenido, admin"
        Sino
            Mostrar mensaje: "Usuario o contraseña incorrectos"
Fin

<script>
document.getElementById('login-form').addEventListener('submit', function(e){
    e.preventDefault(); // Evita recargar la página

    const usuario = document.getElementById('usuario').value.trim();
    const password = document.getElementById('password').value.trim();
    const mensaje = document.getElementById('mensaje');

    // Validación campos vacíos
    if(!usuario || !password){
        mensaje.textContent = "Debe completar todos los campos";
        mensaje.className = "error";
        mensaje.style.display = "block";
        return;
    }

    // Validación login
    if(usuario === "admin" && password === "1234"){
        mensaje.textContent = `Bienvenido, ${usuario}`;
        mensaje.className = "exito";
    } else {
        mensaje.textContent = "Usuario o contraseña incorrectos";
        mensaje.className = "error";
    }

    mensaje.style.display = "block";
});
</script>

</body>
</html>

5️⃣ Script documentado línea por línea

document.getElementById('login-form') → selecciona el formulario.

.addEventListener('submit', function(e){ ... }) → ejecuta la función cuando se envía el formulario.

e.preventDefault() → evita que el formulario recargue la página.

.trim() → elimina espacios al inicio y al final del input.

Validación vacíos → muestra mensaje de error si algún campo está vacío.

Validación login → compara usuario y contraseña con valores “admin / 1234”.

mensaje.textContent → escribe el mensaje en el div.

mensaje.className → aplica estilos CSS de error o éxito.

mensaje.style.display = "block" → muestra el div oculto.
