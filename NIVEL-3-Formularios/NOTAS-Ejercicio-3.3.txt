Ejercicio 3.3 – Encuesta de Opinión
1️⃣ Objetivo del ejercicio

Crear un formulario tipo encuesta donde los usuarios puedan opinar sobre un servicio o producto. Debe incluir:

Nombre del usuario

Correo electrónico

Selección de satisfacción (radio buttons)

Casillas de verificación (checkboxes) de aspectos que más le gustan

Área de comentarios

Botón de envío que valide los campos y muestre un resumen de la encuesta

Se practica:

Inputs text, email, radio, checkbox, textarea

Validación básica con HTML (required)

Interactividad con JavaScript para mostrar resultados

Estilos CSS para presentación

2️⃣ Semántica

<form> → Contenedor del formulario

<label> → Texto descriptivo de cada campo

<input> → Campos de entrada (text, email, radio, checkbox)

<textarea> → Campo de comentarios

<button> → Botón de envío

<div> → Contenedor para resultados y estilos

3️⃣ Pseudocódigo
Inicio EncuestaOpinion
    Mostrar título "Encuesta de Opinión"
    Crear formulario
        Campo Nombre (texto, obligatorio)
        Campo Correo (email, obligatorio)
        Selección de satisfacción (radio, obligatorio)
        Casillas de aspectos favoritos (checkbox, opcional)
        Campo comentarios (textarea, opcional)
        Botón Enviar
    Al enviar
        Validar campos obligatorios
        Mostrar resumen de los datos ingresados debajo del formulario
    Aplicar CSS para centrar y embellecer el formulario
Fin EncuestaOpinion

/* -----------------------------
JavaScript para mostrar un resumen de la encuesta al enviar
----------------------------- */

/* Seleccionamos el formulario por su ID 'encuesta-form' y le agregamos un evento */
document.getElementById('encuesta-form').addEventListener('submit', function(e){
    // e es el objeto del evento, contiene info del submit

    e.preventDefault(); 
    /* 
    preventDefault() evita que el formulario se envíe al servidor
    y que la página se recargue automáticamente. 
    Esto nos permite manejar el envío con JavaScript.
    */

    /* Obtenemos el valor del campo de nombre */
    const nombre = document.getElementById('nombre').value;
    // .value toma el texto que el usuario escribió en el input

    /* Obtenemos el valor del campo de correo electrónico */
    const email = document.getElementById('email').value;

    /* 
    Obtenemos el valor del radio seleccionado (satisfacción)
    querySelector selecciona el primer input que esté marcado con checked
    'input[name="satisfaccion"]:checked' busca dentro de los radios
    el que esté seleccionado
    */
    const satisfaccion = document.querySelector('input[name="satisfaccion"]:checked')?.value;
    // ?.value es "optional chaining", si no hay ninguno seleccionado devuelve undefined

    /* 
    Obtenemos los valores de los checkboxes seleccionados
    Array.from convierte la lista de nodos en un array para poder usar map
    map extrae el valor de cada checkbox
    join(', ') une todos los valores en una cadena separada por coma
    */
    const aspectos = Array.from(document.querySelectorAll('input[name="aspectos"]:checked'))
                        .map(cb => cb.value)
                        .join(', ');

    /* Obtenemos el texto del textarea de comentarios */
    const comentarios = document.getElementById('comentarios').value;

    /* 
    Insertamos dentro del div con ID 'resultado' un resumen en HTML
    innerHTML reemplaza el contenido del div con el texto que construimos
    usando template literals (``) para incluir variables dentro de HTML
    */
    document.getElementById('resultado').innerHTML = `
        <h3>Resumen de la Encuesta</h3>
        <p><strong>Nombre:</strong> ${nombre}</p>
        <p><strong>Correo:</strong> ${email}</p>
        <p><strong>Satisfacción:</strong> ${satisfaccion}</p>
        <p><strong>Aspectos favoritos:</strong> ${aspectos || 'Ninguno seleccionado'}</p>
        <p><strong>Comentarios:</strong> ${comentarios || 'Sin comentarios'}</p>
    `;
    /* 
    ${variable || 'valor por defecto'} significa: si variable está vacía o undefined,
    muestra 'valor por defecto'.
    */
});
</script>
</body>

🔹 Resumen de comandos y conceptos

document.getElementById('id') → Selecciona un elemento del HTML por su ID.

.addEventListener('submit', func) → “Escucha” el evento submit (clic en enviar) y ejecuta la función func.

e.preventDefault() → Evita que la página se recargue cuando se envía el formulario.

.value → Obtiene el contenido de un input o textarea.

document.querySelector('selector') → Selecciona el primer elemento que coincide con el selector CSS.

?.value → Operador opcional que evita errores si el elemento no existe.

document.querySelectorAll('selector') → Selecciona todos los elementos que cumplen la condición y devuelve una NodeList.

Array.from() → Convierte una NodeList en un array real para poder usar métodos como map().

.map(cb => cb.value) → Recorre cada elemento del array y extrae su valor.

.join(', ') → Une los elementos de un array en un string separados por comas.

innerHTML = `` → Inserta contenido HTML dinámico dentro de un elemento.

${variable} → Template literals para incrustar variables dentro de cadenas HTML.






Vamos a hacer la versión ultra comentada, explicando cada símbolo, operador y línea, para que no te quede ninguna duda de cómo funciona este JS de la encuesta:

<script>
/* ==============================================
   JavaScript ultra comentado para la Encuesta
   ============================================== */

/*
  1️⃣ Seleccionamos el formulario completo usando su ID 'encuesta-form'
  2️⃣ Agregamos un "listener" que ejecuta la función cada vez que el usuario hace submit (clic en Enviar)
*/
document.getElementById('encuesta-form')   // Devuelve el elemento HTML <form> con id="encuesta-form"
.addEventListener('submit', function(e){  // 'submit' es el evento de enviar formulario
                                           // La función se ejecuta cada vez que se envía
                                           
    e.preventDefault(); // ⚡ Evita la recarga automática de la página
                        // Sin esto, el formulario se enviaría y la página se refrescaría

    // =====================
    // OBTENER DATOS DEL FORMULARIO
    // =====================

    // 1️⃣ Nombre del usuario
    const nombre = document.getElementById('nombre').value;
    // document.getElementById('nombre') → selecciona el input con id="nombre"
    // .value → obtiene el texto que el usuario escribió

    // 2️⃣ Correo electrónico
    const email = document.getElementById('email').value;

    // 3️⃣ Radio buttons (satisfacción)
    const satisfaccion = document.querySelector(
        'input[name="satisfaccion"]:checked'  // Selecciona el primer radio button marcado con name="satisfaccion"
    )?.value;  // ?. → operador opcional, evita error si no hay ninguno seleccionado
               // .value → devuelve el valor del radio seleccionado

    // 4️⃣ Checkbox (aspectos favoritos)
    const aspectos = Array.from(
        document.querySelectorAll('input[name="aspectos"]:checked')  
        // querySelectorAll devuelve todos los checkbox marcados con name="aspectos"
    )
    .map(cb => cb.value)   // map recorre cada checkbox y devuelve su valor
    .join(', ');           // join une todos los valores en un string separado por comas

    // 5️⃣ Comentarios adicionales
    const comentarios = document.getElementById('comentarios').value;

    // =====================
    // MOSTRAR RESULTADOS EN PANTALLA
    // =====================

    document.getElementById('resultado').innerHTML = `
        <h3>Resumen de la Encuesta</h3>
        <p><strong>Nombre:</strong> ${nombre}</p>
        <p><strong>Correo:</strong> ${email}</p>
        <p><strong>Satisfacción:</strong> ${satisfaccion}</p>
        <p><strong>Aspectos favoritos:</strong> ${aspectos || 'Ninguno seleccionado'}</p>
        <p><strong>Comentarios:</strong> ${comentarios || 'Sin comentarios'}</p>
    `;
    /*
        innerHTML → permite escribir contenido HTML dentro del div con id="resultado"
        `${variable}` → template literal, permite insertar variables dentro de un string
        || 'Texto' → operador OR; si la variable está vacía o undefined, muestra el texto por defecto
    */

}); // Fin de la función que maneja el submit
</script>

    const aspectos = Array.from(document.querySelectorAll('input[name="aspectos"]:checked'))
                          .map(cb => cb.value)
                          .join(', ');
    const comentarios = document.getElementById('comentarios').value;

    // Mostramos el resultado en pantalla
    document.getElementById('resultado').innerHTML = `
        <h3>Resumen de la Encuesta</h3>
        <p><strong>Nombre:</strong> ${nombre}</p>
        <p><strong>Correo:</strong> ${email}</p>
        <p><strong>Satisfacción:</strong> ${satisfaccion || 'Sin seleccionar'}</p>
        <p><strong>Aspectos favoritos:</strong> ${aspectos || 'Ninguno seleccionado'}</p>
        <p><strong>Comentarios:</strong> ${comentarios || 'Sin comentarios'}</p>
    `;
