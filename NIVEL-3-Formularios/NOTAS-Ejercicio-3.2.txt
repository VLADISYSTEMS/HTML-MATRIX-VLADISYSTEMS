Ejercicio 3.2 – Registro de Usuario
1️⃣ Objetivo del ejercicio

Crear un formulario de registro donde los usuarios puedan crear una cuenta ingresando:

Nombre completo

Correo electrónico

Nombre de usuario

Contraseña

Confirmar contraseña

Se practicará:

Inputs de distintos tipos (text, email, password)

Atributos de validación (required, minlength, maxlength)

Estilos CSS para mejorar apariencia y usabilidad

2️⃣ Semántica

<form> → Contenedor del formulario

<label> → Texto descriptivo para cada campo

<input> → Campos de entrada (texto, correo, contraseña)

<button> → Botón de envío

<fieldset> y <legend> → Agrupar campos relacionados (opcional)

<div> → Contenedor para aplicar estilos

3️⃣ Pseudocódigo
Inicio RegistroUsuario
    Mostrar título "Registro de Usuario"
    Crear formulario
        Crear campo Nombre completo (texto, obligatorio)
        Crear campo Correo electrónico (email, obligatorio)
        Crear campo Nombre de usuario (texto, obligatorio)
        Crear campo Contraseña (password, obligatorio, mínimo 6 caracteres)
        Crear campo Confirmar contraseña (password, obligatorio, debe coincidir)
        Crear botón Registrarse
    Aplicar estilos CSS para centrar formulario y mejorar apariencia
Fin RegistroUsuario
