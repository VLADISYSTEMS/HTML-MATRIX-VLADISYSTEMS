1️⃣ Objetivo del ejercicio

Crear una página HTML semántica con CSS integrado, que muestre un menú de restaurante:

Encabezado atractivo con el nombre del restaurante.

Menú organizado en categorías con estilo visual.

Cada plato con nombre, descripción y precio resaltados.

Pie de página con información de contacto.

Página responsiva y visualmente agradable.

2️⃣ Semántica

<header> → Encabezado del restaurante.

<nav> → Barra de navegación (Inicio, Menú, Contacto).

<main> → Contenido principal del menú.

<section> → Categorías (Entradas, Principales, Postres, Bebidas).

<article> → Cada plato individual.

<footer> → Información de contacto.

CSS se aplicará para:

Colores, fondos, tipografía.

Separación de secciones y artículos.

Efectos simples de hover en enlaces.

3️⃣ Pseudocódigo
Iniciar página HTML
    Definir encabezado con nombre del restaurante
    Definir barra de navegación
    Para cada categoría del menú
        Crear sección con título de categoría
        Para cada plato en la categoría
            Crear artículo con nombre, descripción y precio
    Aplicar CSS:
        - Colores, fondos y tipografía
        - Separación de secciones
        - Hover en enlaces
    Definir pie de página con contacto
Fin de página
