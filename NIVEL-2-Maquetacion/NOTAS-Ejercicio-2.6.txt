1️⃣ Objetivo del ejercicio

Practicar el uso de divisiones (<div>) y clases CSS para organizar contenido.

Crear una página con secciones diferenciadas visualmente.

Aplicar CSS para colores, espaciado y responsividad.

2️⃣ Semántica

<header> → Encabezado principal.

<main> → Contenedor de secciones.

<div> → Cada división o bloque de contenido.

<footer> → Pie de página.

Uso de clases CSS para estilizar cada bloque.

3️⃣ Pseudocódigo
Iniciar HTML
    Crear encabezado (<header>) con título de la página
    Crear contenido principal (<main>)
        Para cada división de contenido
            Crear <div> con clase específica
                Agregar título (<h2>) y párrafo (<p>)
                Agregar enlace "Leer más" a ejemplo real
    Agregar pie de página (<footer>) con contacto y derechos
Aplicar CSS
    Estilos generales, colores, bordes, padding
    Responsividad con media queries
Fin

4️⃣ Script limpio (HTML + CSS interno)
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Divisiones HTML</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0; padding: 0;
            background-color: #f0f0f0;
            color: #333;
        }

        header {
            background-color: #ff6f3c;
            color: white;
            text-align: center;
            padding: 20px 0;
        }

        main {
            max-width: 900px;
            margin: 20px auto;
            padding: 0 20px;
        }

        .bloque {
            background-color: white;
            border-radius: 8px;
            padding: 20px;
            margin: 15px 0;
            box-shadow: 0 4px 6px rgba(0,0,0,0.1);
        }

        .bloque h2 {
            color: #ff6f3c;
            margin-bottom: 10px;
        }

        .bloque p {
            line-height: 1.6;
        }

        .bloque a {
            color: #4CAF50;
            text-decoration: none;
        }

        .bloque a:hover {
            text-decoration: underline;
        }

        footer {
            background-color: #ff6f3c;
            color: white;
            text-align: center;
            padding: 15px 0;
            margin-top: 40px;
        }

        @media (max-width: 600px) {
            .bloque {
                padding: 15px;
            }
        }
    </style>
</head>
<body>
    <header>
        <h1>Divisiones HTML El Tigre</h1>
    </header>

        <div class="bloque">
            <h2>Sección 1: Introducción a Divs</h2>

5️⃣ Script documentado (línea por línea)
<!DOCTYPE html>
<!-- Documento HTML5 -->
<html lang="es">
<head>
    <meta charset="UTF-8">
    <!-- Codificación UTF-8 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- Página responsiva -->
    <title>Divisiones HTML</title>

    <style>
        /* Estilos generales */
        body {
            font-family: Arial, sans-serif;
            margin: 0; padding: 0;
            background-color: #f0f0f0; /* Fondo gris claro */
            color: #333; /* Texto gris oscuro */
        }

        /* Encabezado */
        header {
            background-color: #ff6f3c; /* Fondo naranja */
            color: white;
            text-align: center;
            padding: 20px 0;
        }

        /* Contenedor principal */
        main {
            max-width: 900px;
            margin: 20px auto;
            padding: 0 20px;
        }

        /* Bloques con clase */
        .bloque {
            background-color: white;
            border-radius: 8px;
            padding: 20px;
            margin: 15px 0;
            box-shadow: 0 4px 6px rgba(0,0,0,0.1); /* Sombra suave */
        }

        .bloque h2 {
            color: #ff6f3c;
            margin-bottom: 10px;
        }

        .bloque p {
            line-height: 1.6;
        }

        .bloque a {
            color: #4CAF50;
            text-decoration: none;
        }

        .bloque a:hover {
            text-decoration: underline;
        }

        footer {
            background-color: #ff6f3c;
            color: white;
            text-align: center;
            padding: 15px 0;
            margin-top: 40px;
        }

        /* Responsividad */
        @media (max-width: 600px) {

                padding: 15px;

    </style>
</head>
