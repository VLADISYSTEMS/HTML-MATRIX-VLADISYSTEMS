1️⃣ Objetivo del ejercicio

Crear una página web de noticias simple que muestre titulares, subtítulos, párrafos y enlaces, usando la estructura semántica de HTML para organizar correctamente la información y practicar la maquetación básica.

2️⃣ Semántica

<header>: Contiene el título principal de la página.

<nav>: Menú de navegación (opcional en esta página simple).

<main>: Contiene el contenido principal de las noticias.

<article>: Cada noticia individual.

<h1>, <h2>: Titulares y subtítulos.

<p>: Párrafos de texto de la noticia.

<a>: Enlaces a fuentes externas o relacionadas.

<footer>: Información de pie de página.

3️⃣ Pseudocódigo
INICIO
  CREAR página HTML
    AGREGAR <header> con título "Noticias del Día"
    AGREGAR <main>
      PARA CADA noticia
        CREAR <article>
          AGREGAR <h2> como título de la noticia
          AGREGAR <p> como contenido de la noticia
          AGREGAR <a> para enlace externo
      FIN PARA
    AGREGAR <footer> con copyright
FIN
